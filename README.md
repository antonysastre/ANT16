# ANT16 Computer

A fun summer project 2025 with the goal to build a simple 16-bit computer that can hold a CPU architecture, an Assembly language, a simple OS, a higher level programming language and some games build in that language.

## Naming

**Ant16: The Computer.**

- AntDL: The hardware description language (HDL) for the chipsets. 
- AntXU: The instructions set i.e. CPU architecture.
- AntASM: The assembly syntax/language.
